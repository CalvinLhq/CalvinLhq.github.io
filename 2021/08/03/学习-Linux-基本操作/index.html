学习-Linux-基本操作

两眼抹黑啥也不知道，那就从安装系统开始吧！

  1. 安装 Linux 系统
    话说是要安装系统，但要如何开始呢？从哪里找来系统？找来什么系统？又要安装到哪里去呢？这些都是我要攻破的问题。

    • 安装 Linux 系统之前准备工作：
      对于新手来说，练习一般都是在个人电脑上完成的，但是直接在电脑上重装系统，将原有系统覆盖显然是行不通的。
      比较合理的方式是，在电脑上安装虚拟机环境，并在环境内练习安装 Linux 系统的虚拟机。

      • 安装 VMware Workstation 虚拟机
        步骤很简单，到 VMware 官网去下载就好了。
      • 下载合适的系统镜像
        什么是镜像？我们一般称其为光盘镜像。你可以把一个镜像理解成一张光盘，一张没有实体的以文件形式存在的光盘，里面封存着一个操作系统的所有信息。镜像可以从网上各种渠道下载，我从网上选择了 CentOS 7.6 版本的镜像。CentOS 7.6 是众多 Linux 发行的操作系统之一，容易上手，方便安装。有了镜像和 VMware Workstation，就可以开始安装了。

    • 开始安装
      在 VMware Workstation 内选择新建虚拟机，然后按照步骤和提示创建即可。创建完成后会开始初始化程序，黑底白字的命令行开始在眼前奔腾，不需要去顾虑上面的内容。等待一会后进入 CentOS 安装界面，大部分设置项都不用改动，其中值得一提的是硬盘的分配。

      • 硬盘分配
        这里我了解到了 Linux 系统万物皆文件的属性，稍微学习了一下各主要文件目录的作用。之后暂且根据网络上他人推荐的配置完成了设置。
      • 设置 root 用户密码
        root 用户是权限最高的管理员，所以不要忘记 root 用户的密码。忘记密码的话也有方法可以重置，方法繁琐，可以在网络上找到方法。设置完密码后点击安装就完成了。

    • 配置网络
      从这一步开始，难度开始上升。
      操作系统安装完毕之后以 root 用户登入到系统中，进入终端，也就是命令行，然后输入 ifconfig 或者 ip a 后回车来查看网卡配置情况。找到正在使用的网卡后，使用 vim 编辑器 修改 /etc/sysconfig/network-scripts/网卡名 路径下的文件。具体步骤在 虚拟化与远程连接 中会更详细的介绍。

  1. 学习 Linux 命令行指令
    一般情况下，我们的工作都是在命令行模式下进行的。可以使用 systemctl set-default multi-user.target 命令切换到命令行模式，这样卡机以后就能进入到黑底白字的命令行了。

    • 目录管理
      在 Linux 系统中工作必不可少的要在众多文件目录中穿梭。所以所以除了知晓主要目录的作用外，还要掌握如何创建，移动，命名，授权目录的方法。
    • 文件管理

以上都是最为基础的文件指令，相当于鼠标于图形化界面。Linux 学习是一个漫长的积累过程，设计者们也考虑到了指令繁多可能会导致的使用困难，于是也开发了一些帮助我们使用 Linux 的指令：想要了解的指令 --help 或 man 想要了解的指令。这两个指令会提供给我们想要了解的信息，前者提供的信息更精简，后者更详细。在学习期间要善于去运用这两个指令。
实用的命令还有太多太多，重启系统的 reboot，管理软件包的 tar 和 yum，用来查找文件的 find 等。都需要根据具体的工作内容进行调整学习。
